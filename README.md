# aimc-mainframe-mentoring
# 🧠 AIMC Mainframe Mentoring

This repository documents how to understand and work with mainframe systems.

It is not focused on COBOL syntax or implementation details,
but on how system behavior can be interpreted, understood, and learned.

---

## 🎯 Purpose

The goal is to make complex system behavior:

- understandable
- explainable
- teachable

---

## 🧠 Core Concept

- COBOL → defines what the system does
- AIMC → shows what happened during execution
- REXX → makes decisions explicit

---

## 🧭 Learning Path

1. System thinking  
   → `knowledge-behind/system-thinking.md`

2. AIMC mentoring concept  
   → `knowledge-behind/mentoring/aimc-ai-mentoring-concept.md`

3. Decision patterns  
   → `knowledge-behind/mentoring/decision-patterns/`

4. Examples (REXX + ISPF panels)  
   → `examples/`

---

## ✅ What this repo is

- A mentoring framework
- A decision-pattern library
- A structured knowledge base

---

## ❌ What this repo is not

- Not a COBOL tutorial
- Not system documentation
- Not a framework

---

## 🚀 Vision

Turn implicit system behavior into:

- explicit rules
- reusable patterns
- structured learningaimc-mainframe-mentoring
